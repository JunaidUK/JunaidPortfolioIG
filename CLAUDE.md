# CLAUDE.md

This file provides guidance to Claude Code when working with code in this repository.

## Project Overview

This is a FHIR Implementation Guide project for a health tech company owned by Junaid Siddiqui, currently in early conceptual development. The project contains markdown documentation for FHIR concepts, example FHIR resources written in FSH, and sales material for selling Junaid Siddiqui's services. The static HTML is hosted on Github Pages. 

### Documentation Structure
- `output/` - Directory intended for HTML outputs generated by the `_genonce.sh` script.
Ignore this directory.
Do not run any commands against this directory, the contents are meant to be generated dynamically.
- `target/` - Empty directory intended for generated FHIR Canonical resources

## Development Scripts
- `sushi build .` - Builds the Implementation Guide from the root directory
- `_updatePublisher.sh` - Downloads the Implementation Guide Publisher 
- `_genonce.sh` - Generates HTML outputs for Implementation Guide
